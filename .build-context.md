## Key Decisions
- Provider pattern with auto-detect probing known ports because user needs any-backend support
- SQLite for hypothesis dedup because hash lookups need to be fast
- Target model staleness = 14 days (BountyHound proven default)
- 3-layer validation (not 4) because deep by-design checks need Claude-scale reasoning
- Scope defaults to wildcard of root domain when no explicit scope set
- Keep ReAct prompt-based tool calling for max cross-backend compatibility
